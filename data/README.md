# Supplementary datasets

Data underlying the manuscript *"Designing an Automated, Quality-Gated Framework
for High-Throughput Molecular Docking and Localized Pharmacokinetic Profiling."*
All files here are the exact inputs/outputs of the reported runs.

> **Provenance & versions:** `hivpr_benchmark/versions.txt` records the *actual*
> software versions used (AutoDock-GPU v1.6, RDKit 2026.03.3, ADMET-AI 2.0.1).
> Charges are Gasteiger (Open Babel), not Kollman — MGLTools was not used.

## `hivpr_benchmark/` — DUD-E HIV-1 protease retrospective benchmark

Target 1HXB, native inhibitor **ROC** (saquinavir; single altloc). Generated by
`validation/benchmark_hivpr.py` with `config/benchmark_hivpr.json`.

| File | Description | Manuscript ref |
|------|-------------|----------------|
| `actives_62.smi` | 62 DUD-E HIVPR actives (subset of 536) | §2.8 / §3.4 |
| `decoys_188.smi` | 188 decoys, MW-quartile stratified from 35,750 (seed 42) | §2.8 / §3.4 |
| `hivpr_benchmark_validation_results.xlsx` | Ranked 250 compounds + Metrics/Control/Subset sheets | Table (benchmark), Fig 5 |
| `native_control.dlg` | Native ROC redocking log (100 GA runs) | §2.8 redock control |
| `native_control_top1.pdb` | Lowest-energy redocked ROC pose | Fig 2 (control) |
| `config_snapshot.txt` | Exact parameters used | §2.5 |
| `versions.txt` | Software provenance (actual versions) | §2.1 |
| `timing_log.txt` | 30.7 min runtime breakdown + cooldown config | §3.4 timing |

**Headline result:** ROC-AUC = 0.806 (95% CI 0.738–0.872); redock control
RMSD 0.63 Å, ΔG −13.88 kcal/mol; top-15 = 15/15 actives.

## `wrn_screen/` — WRN 9S18 prospective screen

| File | Description | Manuscript ref |
|------|-------------|----------------|
| `hro761_analogs_100.smi` | 100 synthetic HRO761 analogs, canonical SMILES (RDKit) derived from the docked 3D conformers | §3.1–3.3 |
| `9S18_lowest_energy_consensus.csv` | Consensus ranking (3 replicates, seeds 42/43/44; mean/median/σ dG, rank-stability) via `pose_selection=lowest_energy` | §3.2 |

## Reproduce

```bash
export DOCKSUITE_DATA=/path/to/work        # portable data root
python validation/benchmark_hivpr.py       # regenerates hivpr_benchmark/*
python scripts/run_screen.py --config config/screen_wrn.json   # WRN screen (needs the 9S18 structure)
```

## Not included (intentionally)
Raw per-ligand `.dlg`/`.pdbqt`/AutoGrid `.map` files (hundreds of MB) are
regenerated by the pipeline and are excluded via `.gitignore`. Only the native
control `.dlg` (766 KB) is kept as the redocking-validation artifact.
